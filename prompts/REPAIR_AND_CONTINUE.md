# Prompt — réparer puis continuer

Le dépôt contient probablement un travail incomplet. Identifie le dernier état réellement fonctionnel à partir de Git et des tests, reproduis chaque défaut, corrige les causes racines, ajoute les tests de non-régression, puis reprends le sprint actif sans étendre son périmètre. N'avance pas le statut du sprint avant une quality gate complète et un scénario démontrable.
