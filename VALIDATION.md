# Validation du kit généré

Validé dans l’environnement de génération :

- JSON et structure du dépôt ;
- installation NPM reproductible ;
- lint Web ;
- test Vitest ;
- type-check Vue/TypeScript ;
- build Vite de production.

Non exécutable dans l’environnement de génération faute d’outils installés : .NET, Docker et Gradle/Android SDK. Ces validations constituent précisément le périmètre restant de SPRINT-00.
