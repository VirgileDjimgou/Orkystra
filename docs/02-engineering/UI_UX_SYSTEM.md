# Système UI/UX

## Web

- layout professionnel avec sidebar compacte, barre supérieure et contenu respirant ;
- Bootstrap utilisé comme fondation, pas comme identité visuelle brute ;
- tokens CSS pour espace, rayon, typographie et états ;
- tableaux avec filtres, colonnes stables et actions contextuelles ;
- carte et liste synchronisées ;
- responsive desktop/tablette, mobile Web secondaire.

## Android Conducteur

- grandes zones tactiles ;
- contraste élevé ;
- nombre minimal d'étapes ;
- état de synchronisation toujours visible ;
- mission courante accessible immédiatement ;
- photos/signature guidées ;
- aucune animation bloquante.

## États obligatoires

Loading, empty, error récupérable, erreur bloquante, offline, synchronisation en attente, succès, permission refusée.
