# Passage du MVP au produit commercial

## Pilote

- 1 à 3 entreprises ;
- 5 à 20 véhicules chacune ;
- contrat pilote limité ;
- support horaires définis ;
- intégration par CSV ou un seul fournisseur GPS ;
- données réelles uniquement après revue RGPD et sécurité.

## Preuves de valeur

- réduction des appels ;
- temps de clôture d'une mission ;
- taux de preuves complètes ;
- incidents détectés ;
- utilisateurs actifs ;
- volonté de payer.

## Avant vente générale

- nom et marque vérifiés ;
- CGU/DPA ;
- sauvegardes testées ;
- MFA ;
- surveillance et alertes ;
- processus incident ;
- rétention et export ;
- facturation SaaS ;
- documentation d'onboarding.
