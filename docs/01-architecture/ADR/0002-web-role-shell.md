# ADR-0002 — Une application Web fondée sur les rôles

## Statut

Accepté.

## Décision

Administrateur et Opérateur utilisent une seule application Vue 3 avec navigation et autorisations par rôle.

## Raisons

Éviter la duplication de composants, de modèles et de pipelines. Les écrans peuvent conserver des shells distincts sans créer deux produits techniques.
