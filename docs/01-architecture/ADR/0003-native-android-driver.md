# ADR-0003 — Application Android native Conducteur

## Statut

Accepté.

## Décision

Kotlin et Jetpack Compose, architecture mono-Activity, UDF, ViewModel, Repository, Room et WorkManager.

## Raisons

Le conducteur a besoin d'un comportement fiable hors ligne, de caméra, de localisation, de tâches de synchronisation et d'une ergonomie mobile dédiée.
